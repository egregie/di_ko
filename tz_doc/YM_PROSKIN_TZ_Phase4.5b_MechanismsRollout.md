## Блок A — Механизм/анатомия по трём темам
Для каждой темы определить плейсхолдер-слоты в её spec и заменить реальными ассетами. Источник:
**Servier Medical Art (CC-BY 4.0)** — перекрас под бренд разрешён; если точной схемы нет —
упрощённая **source-grounded SVG**. Анатомию слоёв кожи (общая для тем) — **переиспользовать**
`skin_layers_turnover.svg`, не плодить копии.

Целевой инвентарь (сверить с реальными слайдами; шаги — только из фактов темы):
- **vitamin_c:**
  - синтез коллагена (аскорбат — кофактор гидроксилаз → коллаген I/III) ← fact_0021, fact_0030/0046;
  - схема проникновения/pH (pH<3.5, макс. абсорбция при 20%) ← fact_0028/0044, fact_0029/0045
    (это концепт-схема, не биология — ниже риск).
- **niacinamide:**
  - барьер/синтез церамидов ← fact_0024;
  - подавление переноса меланосом (меланоцит→кератиноцит) ← fact_0034 *(высокий риск точности)*.
- **exfoliants:**
  - десквамация/десмолиз рогового слоя (BHA разрывает межклеточные соединения) ← fact_0049;
  - гистологический эффект AHA на дерму (толщина/коллаген) ← fact_0048 (можно опереть на общую
    анатомию слоёв кожи).

`[GUARD]` **diagram↔факт:** каждый изображённый шаг механизма поддержан фактом графа темы. Нет
факта на шаг — шаг не изображаем (или сперва собрать факт через гейт). Не «биологически
правдоподобно» — а подтверждено.
`[GUARD]` Лицензия: только Servier CC-BY (с реальным image ID в provenance) или source-grounded SVG.
Запрещены CC-BY-SA / NC / BioRender-без-Industry / «без явной лицензии». Provenance + attribution
обязательны (P019).
