## Блок A — Фикс схемы `rar_rxr_mechanism.svg`
1. Центрировать весь внутриядерный узел (димер RXR–RAR + бокс «RARE Element» + нить ДНК) внутри
   пунктирной ядерной арки по обеим осям — сейчас смещён влево-вниз.
2. `[GUARD]` Развести боксы **RXR** и **RAR** — они перекрываются; поставить рядом как
   гетеродимер с зазором; метка «RXR» должна быть видна целиком (сейчас обрезана до «RX»).
3. Сдвинуть бейдж лиганда «L», чтобы он не налезал на надпись «RAR».
4. Центрировать иконку RA (пентагон) внутри круга цитоплазмы.
5. Уменьшить бокс «RARE Element» до размера текста + паддинг; провести нить ДНК по центру через
   него; выровнять/почистить красные направляющие стрелки (по центру линии).
6. Выровнять подписи секций (ЦИТОПЛАЗМА, ЯДРО/NUCLEUS), единые отступы.

## Блок C — Diagram Layout Checklist (переиспользуемый, применять и к Блокам A/B, и ко ВСЕМ новым схемам 4.5b)
- `viewBox` плотно охватывает контент + равномерный паддинг (≥24px) со всех сторон; ничто не
  касается и не обрезается у края.
- Нет перекрытий фигур/подписей; смежные элементы (димеры, соседние боксы) разнесены с зазором.
- Текст влезает в свой контейнер (мерить; перенос или уменьшение); подписи выровнены единообразно.
- Иконки центрированы внутри своих контейнеров.
- Центральный контент центрирован внутри обрамления (арки, рамки).
- Боксы по размеру текста + паддинг, не раздуты.
- Zero Black / палитра бренда / Arimo держатся.
