## Блок B — Двойной рендер из одного `slide-spec`
`[GUARD]` Оба рендера потребляют ИДЕНТИЧНЫЙ `slide-spec`; цвет/шрифт только из токенов; расхождений нет.
1. `render_html.py` (есть) → PDF через Playwright (Chromium print), 16:9.
2. **Новый** `render_pptx.py` (python-pptx): тот же `slide-spec` + токены → редактируемый `.pptx`.
   - координаты блоков по `layout`; Arimo встроен; картинки кадрировать (cover), не растягивать;
   - Zero Black; типошкала из токенов.
3. Выход: `06_render/out/deck_retinoids_v2.pdf` и `deck_retinoids_v2.pptx`.
