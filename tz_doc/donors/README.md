## Требуется (Definition of Ready для Phase 8.1 Block A)

- [ ] `hd_1.*` — одобренный референс-образец деки **Постакне** (PDF или PPTX). Из него извлекается геометрия слотов. PPTX предпочтительнее (нативные позиции shape'ов), PDF — тоже годится (рендер + измерение bbox).
- [ ] `design_concept.*` — описание/примеры желаемого стиля (PDF/DOCX/MD/изображения).
- [ ] Утверждение **ровно 2 каркасов** (Ретиноиды, Постакне), 13–20 слайдов каждый — словесно или внутри design_concept.

## Чего НЕ класть
- `tpl1` / `tpl2` — НЕ парсить (грязная геометрия, явный запрет в roadmap).
- Наши собственные отрендеренные деки из `06_render/out/` — это выход, не доноры.
